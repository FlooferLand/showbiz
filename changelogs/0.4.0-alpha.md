_NOTE:_ This release adds a ton of features, but it might have broken existing ones.

If you find any bugs, please report them on our GitHub / Discord server.

### General
- Added the programmer block and very early show recording
- Added a work-in-progress monitor block and video playback
- Added a basic server-side player permission system and improved the config system
- Added a search function to the bot select screen
- Added a bit picker widget to remove the need for players to manually look up bits
- Added a plush system + Mitzi plush
- Added a work-in-progress multi-bitchart support system
- Added a working cymbal for Rolfe
- Added an effect type to bit blocks to disable pneumatic cylinder sounds for specific movements
- Added ffmpeg support to allow you to use any audio format when selecting show audio
- Added Spanish/Mexican translations _(work of Snowvy)_
- Added partial Romanian translations
- Added autocompletion for the `/showbiz bitmap` command
- Added directory support for show files
- Reworked the old connection system to use dedicated packets. Fixes the very bad lag that happened in multiplayer
- Made the Greybox no longer output audio
- Made the reel-to-reel able to face any direction
- Fixed a ton of audio bugs. Pausing while show audio is playing should work way smoother now.
- Fixed an error regarding addon loading where it reported the addon manifest was broken rather than the `bots.toml`
- Fixed inconsistent rotations and positions for the inventory items, and broken missing textures
- Fixed broken audio formats remaining broken forever
- Fixed show files clashing when they have similar names
- Fixed a ton of random bugs
- Improved bot rendering performance
- Improved Dook's model _(work of xion_no.i/)_
- Improved Beach Bear's model _(work of xion_no.i/)_
- Improved Rolfe's model
- Removed the `/showbiz reelupload` command
