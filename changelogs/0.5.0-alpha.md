### Addon
The old bitchart names have been deprecated this update, and are planned to be removed sometime by 0.6.X or later

Please make sure to update all of your addons to use numerical bit IDs _(16td, 166, etc)_ to prevent your addons from no longer working in the future

You can view the bitcharts [here](https://github.com/FlooferLand/bizlib/tree/main/src/main/resources/bitmaps).
Using the Showbiz VSCode extension speeds up the process

### General
- Added the Wolf-Pack 5, and a `wp5` bitchart
- Fixed a bug with Mitzi's pompoms teleporting all over the place when using shaders
- Updated the bitchart names using more accurate data
